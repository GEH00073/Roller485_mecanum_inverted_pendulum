# コントローラー操作

Atom JoyStick コントローラーは、一般的な RC 送信機の Mode 2 配置に合わせています。

## スティック割り当て

- 左スティック上下: Throttle
- 左スティック左右: Rudder
- 右スティック上下: Elevator
- 右スティック左右: Aileron

本機は飛行 RC ではありませんが、メカナムホイールの動きを表すときにこの名称が直感的で分かりやすいため、RC のスティック名をそのまま使用しています。

## 操作前の確認

操作を始める前に、本体の周囲に十分な空間があることを確認してください。初回確認時や調整中は、必ず車輪を浮かせた状態で動作を確認してください。
