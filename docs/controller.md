# コントローラー操作
　本体を垂直に立ててバランスがとれたところでコントローラーの右上ボタンを　押すと起動し自立します。

## スティック割り当て
　Atom JoyStick コントローラーは、一般的な RC 送信機の Mode 2 配置　に合わせています。

- 左スティック上下: ロールバイアス調整　上:前に傾ける　下:後に傾ける
- 左スティック左右: 旋回　左:左回り　右:右回り
- 右スティック上下: 前進後進　上:前進　下:後進
- 右スティック左右: 左右移動　左:左移動　右:右移動

本機は飛行 RC ではありませんが、メカナムホイールの動きを表すときにこの名称が直感的で分かりやすいため、RC のスティック名をそのまま使用しています。

## 操作前の確認

操作を始める前に、本体の周囲に十分な空間があることを確認してください。初回確認時や調整中は、必ず車輪を浮かせた状態で動作を確認してください。
